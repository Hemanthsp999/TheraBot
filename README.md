# How to contribute ?

1. Fork the repo

2. clone the forked repo in local machine

3. cd "project_repo"

4. install dependency

```bash
npm install
```

5. exec the frontend

```bash
npm run dev
```

For Backend:

```bash
1. cd "backend_dir"
2. pip freeze > requirements.txt
3. Install the requirements
4. python servermanage.py runserver
```
